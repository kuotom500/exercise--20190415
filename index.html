Flycan Responsive Web Design

這個RWD教學課程有大量的實作範例讓同學練習，（RWD 中文翻譯：響應式網頁設計，自適應網頁設計，回應式網頁設計）都市針對iphone 和 Android 智慧型手機還有Ipad和Android平板電腦鎖量身定做的網頁版型設計案例，RWD教學主要會使用 HTML5 語法作為網頁的主結構，搭配 CSS3 新語法來排版，對於不同尺寸大小的手機或是平板電腦，設計彈性伸縮的響應式網頁版型，可以讓同一套網頁在電腦上正常瀏覽，遇到平板電腦時會自動轉換成適合平板電腦大小的版型，遇到手機時也會再自動轉換成適合手機大小的版型，做到真正的跨平台網頁設計，最後還會提到 CSS3 新的動態網頁效果...等。
